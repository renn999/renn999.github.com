Ext2 Installable File System For Windows

這是一個可以讓Windows讀取Linux的EXT2或EXT3的分個區工具  最新版已經到了1.11版其中有不少的更新

官方網站 可以到裡面看一下

主要功用及支援：

  1. 支援NT 4.0, Windows 2000, Windows XP, Windows 2003 and Windows Vista
  2. 支援32bit x86平台以及64bit x64平台
  3. Includes drivers with a digital signature for Windows Vista x64(我不知道這啥功能 因為沒用Vista...= =)
  4. 支援所有讀寫移動...等功能
  5. UTF-8支援(我認為這是最重要的功能...XDDD)
  6. 2 GBytes大檔案支援(不大清楚 請看官方FQA)
  7. 完整的plug-n-play功能 當磁碟被移除時代號也會被移除
  8. Supports use of the Windows mountvol utility to create or delete drive letters for Ext2 volumes [new] (except on Windows NT 4.0). This is useful for scripts. (Please read the FAQ section, too.)
  9. A global read-only option is provided [new].
  10. File names that start with a dot "." character are treated as hidden [new].
  11. Supports GPT disks if the Windows version used also does [new].
  12. Paging files are supported. (A paging file is a file "pagefile.sys", which Windows swaps virtual memory to.) Users may create paging files at NT's control panel at Ext2 volumes.
  13. Specific functions of the I/O subsystem of NT: Byte range locks, notification of changes of directories, oplocks (which are required by the NT LAN manager for sharing files via SMB).

以上部份 主要功能有翻過了 剩下的都是懶的翻...主要來源來自官方網站

我覺得有LINUX雙系統的人裝上這個後真的磁區的可用性大增 而且有已經支援UTF-8了

在ext2上面的中文字都可以正常顯示以及複製到windows底下 真的是雙系統必裝的...XDDD
