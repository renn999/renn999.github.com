Ubuntu 9.04 發佈

今天剛好是Ubuntu9.04發佈的日子 所以說就想說升級來試試看的心態下去升級了 整體讓人感覺上對於速度的提昇有不少的改進 當然我就順道把我的桌機和我就順道把我的桌機和Serverr一起升級上去了

當然最直覺的感覺開機速度提昇不少 還有一些的的小地方感覺上沒有這麼頓了 登入畫面變得比較酷 但是目前還有去測試Ext4怎樣 這應該是晚一點的事情了...

這裡就來寫寫升級的方式好了

  • 圖形介面的升級方式 其實是還蠻簡單的 首先先備份一下source.list
    $ sudo cp /etc/apt/sources.list /etc/apt/sources.list.intrepid
    之後再直接
    $ sudo update-manager -d
    應該就會出現提示有新版本的視窗 然後點下升級就好了
  • 文字介面的升級方式 首先先也是備份一下source.list
    $ sudo cp /etc/apt/sources.list /etc/apt/sources.list.intrepid
    然後
    sudo apt-get install update-manager-core sudo do-release-upgrade
    這樣就可以升級上去了
