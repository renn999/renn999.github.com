    Linux音訊轉檔(一)

    再這裡把幾個常用的有損及無損音樂壓縮整理一下，以及他們在Linux底下如何轉檔所作的整理，基本上無損壓縮壓縮的話不外乎看到的就是tta(TrueAudio) ape(Monkey’s Audio) flac(Free Lossless Audio Codec) wv(Wavpack) tak((T)om’s lossless (A)udio (K)ompressor)，有損的話Mp3(MPEG-1 Audio Layer 3) Ogg(Ogg Vorbis) AAC，基本上比較流行的格式不外乎上面幾種

    無損壓縮：

    tta(TrueAudio)：

    • Downlads：
      官方網站：TrueAudio
      原始碼下載：Downloads
      debPackage：Downloads(UnOffical)
    • Encode：

      [-o …]內為可加可不加，不加為為輸出原始檔名的tta file
    • Decode：
      [-o …]內為同Encode，更多請詳閱ttaenc -h

    ape(Monkey’s Audio)：

    • Downlads：
      官方網站：Monkey’s Audio Codec(.APE) non-win32 Port
      原始碼下載：Downloads
      debPackage：Downloads(UnOffical)
    • Encode：
    • Decode：
      更多請直接詳閱mac

    flac(Free Lossless Audio Codec)：

    • 安裝：
    • Encode：
      [-o …]內為可加可不加，不加為為輸出原始檔名的flac file
    • Decode：
      [-o …]內為同Encode，更多請直接詳閱man flac

    wv(Wavpack)：

    • 安裝：
    • Encode：
      [-o …]內為可加可不加，不加為為輸出原始檔名的wv file，更多請直接詳閱man wavpack
    • Decode：
      [-o …]內為同Encode，更多請直接詳閱man wavunpack

    tak((T)om’s lossless (A)udio (K)ompressor):(這是在Linux底下最麻煩的格式了)

    • Downlads：
      官方網站：(T)om’s lossless (A)udio (K)ompressor
      下載：Download
    • 安裝：
    • Encode：
      [outfile]內為可加可不加，不加為為輸出原始檔名的tak file，更多請直接詳閱wine takc
    • Decode：
      [outfile]內為同Encode
